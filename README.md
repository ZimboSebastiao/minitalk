# minitalk
O objetivo deste projeto é codificar um pequeno programa de troca de dados utilizando sinais UNIX.


## Servidor

1. Inicia
2. Mostra PID
3. Entra em loop
4. Espera sinais
5. Reconstrói bits
6. Imprime caractere

## Cliente

1. Recebe PID
2. Lê string
3. Converte char → bits
4. Envia bit por bit
